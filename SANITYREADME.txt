## 🗂️ Code Files & Examples

### Configuration
- **astro.config.mjs** - Astro config with Sanity integration
- **package.json** - Updated with Sanity scripts
- **.env.example** - Environment variables template

### Sanity Studio
- **sanity/** - Complete Sanity Studio configuration
  - **sanity.config.ts** - Studio configuration
  - **schemas/** - Content type definitions
    - **blogPost.ts** - Blog post schema
    - **lookbookItem.ts** - Portfolio item schema
    - **index.ts** - Schema exports

### Integration Code
- **lib/** - Helper functions
  - **sanity.ts** - Sanity client setup
  - **sanityQueries.ts** - Blog post queries
  - **lookbookQueries.ts** - Lookbook queries

### Components
- **PortableText.tsx** - Rich text renderer component

### Examples
- **blog-sanity.astro** - Example blog listing page
